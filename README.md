# Como executar o projeto

```bash
# clonar repositório
git clone https://github.com/alcemirjunior/desafio-tunts.git
```

```bash
# executar o projeto
./mvnw spring-boot:run
```

```bash
# efetuar a requisição
curl http://localhost:8080/api/v1/tunts-challenge
```
